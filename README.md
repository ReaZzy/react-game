# react-game
Deploy -  https://reazzy.github.io/react-game/#/

# HOW TO START
1. Download
 - ``` git clone https://github.com/ReaZzy/react-game.git ```
2. Install npm modules
 - ``` npm install ```
3. Start app
 - ``` npm run start ```

After start you can open react-game by link http://localhost:3000/
